O que é algoritmo

Um algoritmo é um conjunto de instruções passo a passo que resolve um problema ou realiza uma tarefa. Essas instruções são claras, organizadas e finitas – ou seja, elas precisam ter um começo, meio e fim.

Para que serve

Exemplos:

  • Um caixa eletrônico usa um algoritmo para verificar se você tem saldo antes de liberar o saque.
  • O Instagram usa algoritmos para decidir quais posts mostrar primeiro para você.
  • Aplicativos de transporte usam algoritmos para calcular a melhor rota.
  • Um jogo usa algoritmos para controlar os movimentos dos personagens e do jogador.

Diferença entre algoritmo e programa:

Algoritmo: É o plano lógico, o passo a passo.

Programa: É o algoritmo escrito em uma linguagem que o computador entende.

O algoritmo é como a receita de bolo escrita em papel.

O programa é o bolo pronto feito por um robô que seguiu essa receita.

Etapas de criação de um algoritmo

  • Entender o problema: Saber o que você quer resolver.
  • Planejar os passos: Pensar na sequência lógica de ações.
  • Escrever o algoritmo: Organizar as ações em ordem, como se fosse uma receita.
  • Testar: Ver se ele funciona com diferentes valores ou situações.
  • Melhorar (se precisar): Tornar mais rápido, mais simples ou mais claro.
